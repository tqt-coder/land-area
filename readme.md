### Run project:
1. Frontend:
- Move to frontend folder
    ```
        cd frontend
    ```
- Intall libraries:
    ```
        npm i
    ```
- Run frontend
    ```
        npm start
    ```
2. Backend:
- Move to backend folder:
    ```
        cd backend
    ```
- Install libraries:
    ```
        pip install -r requirements.txt
    ```
- Run backend:
    ```
        flask run
    ```
3. Database mysql
    Script in path: /mysql/sql.sql

### Run with Docker
1. Install images and run images
    ```
        docker-compose up
    ```
2. Stop and kill services:
    ```
        docker-compose down
    ```
